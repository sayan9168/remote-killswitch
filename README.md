# Remote-KillSwitch 🛡️

A lightweight Python library to remotely control and deactivate your code execution. This is ideal for freelancers or developers who want to maintain control over their work, ensuring that code only runs when allowed by a remote configuration file.

## Features ✨
- **Remote Control:** Deactivate your script from anywhere by changing a JSON file.
- **Easy Integration:** Add protection in just 3 lines of code.
- **Custom Messages:** Display a custom message to the user when deactivated.
